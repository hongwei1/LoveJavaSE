      Featured snippet from the web

      认识是认知知识，即人脑反映客观事物的特性与联系、并揭露事物对人的意义与作用的思维活动。 从广义上讲，认识包含人的所有认知活动，即为感知、记忆、思维、想象、语言的理解和产生等心理现象的统称。 认识是一种信息加工过程，可以分为刺激的接收、编码、存储、提取和利用等一系列阶段。 从狭义上讲，认识有时等同于记忆或思维。
